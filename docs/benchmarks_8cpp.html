Detailed Description

Implementation of shared benchmark utilities and global configuration.

Provides:

  • Global configuration: sample counts and thread count
  • Result serialization: save benchmark results to CSV files
  • Shared infrastructure for all benchmark types

Benchmark Components:

  • Integration benchmarks: src/apps/benchmarks/integration_benchmarks.cpp Tests convergence and accuracy of MC, IS, MCMC methods
  • PSO benchmarks: src/apps/benchmarks/pso_benchmarks.cpp Performance testing on sphere, Rastrigin, Rosenbrock functions
  • GA benchmarks: src/apps/benchmarks/ga_benchmarks.cpp Genetic algorithm with population visualization

Sample Configuration:

  • Logarithmic scale: 10k, 50k, 100k, 500k, 1M samples
  • Enables convergence analysis: E[error] ∝ 1/√N
